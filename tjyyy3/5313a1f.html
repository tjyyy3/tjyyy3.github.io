【题解】283. 移动零 - 力扣（LeetCode）

1. 题目描述

给定一个数组 nums，编写一个函数将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。

请注意 ，必须在不复制数组的情况下原地对数组进行操作。

示例 1:

示例 2:

提示:

  • 1 <= nums.length <= 104
  • -231 <= nums[i] <= 231 - 1

2. 解题思路

  1. 使用指针。遍历整个数组，每当遇到一个0就使用指针找到它以后的第一个不为0的数，并交换位置。（代码参考第三部分1）
  2. （官方）使用快慢指针。定义两个指针left=0,right=0，right=0是快指针，用来遍历数组每一个元素，left=0是慢指针，用来定位非0元素。
    当right指向的元素非0时一切正常，执行swap操作（相当于自己与自己交换，什么都没有发生），并进行right++, left++；
    当right指向的元素为0时，只进行left++，此时left会定位0所在的位置，等下一轮遇到非0元素时，就能够将0与非0元素进行互换了~

3. 参考代码

  1. class Solution {
public:
    void moveZeroes(vector<int>& nums) {
        int n = nums.size();
        for(int i=0;i<n;i++)
        {
            if(nums[i]==0) // 说明应该找离它最近的非0元素交换
            {
                int j = i+1;
                while(j<n && nums[j]==0)j++;
                if(j<n && nums[j]!=0) swap(nums[i], nums[j]);
                else 
                    break;
            }   
        }
        for(int i=0;i<n;i++)
            cout<<nums[i]<<" ";
    }
};
