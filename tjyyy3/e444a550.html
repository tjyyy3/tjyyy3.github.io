【JAVA】java中各种实体对象缩写含义

发表于2025-01-23|更新于2025-12-27
|字数总计:191|阅读时长:1分钟|阅读量:

1. pojo(Plain Old Java Object)

pojo指普通的Java对象，是一个简单的Java类。

pojo通常用于封装数据，有getter和setter方法，用于对属性进行读写。

2. DTO(Data Transfer Object)

DTO是数据传输对象，用于将数据库中的数据转换为前端需要的格式，方便前后端之间的数据交互。

2.1 适用场景

3. DAO(Data Access Objects)

DAO是数据访问对象，用于封装数据访问逻辑，负责与数据库进行交互。

4. VO(Value Object)

VO是值对象，用于封装不同的数据属性，方便前端页面的显示和交互；也是用于数据传输的对象，类似DTO，但是VO更专注于视图层的数据展示（前端）。
